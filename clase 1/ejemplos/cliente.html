Cliente HTTP — Clase 1

Hace peticiones a server.php con distintos métodos HTTP y muestra la respuesta.

GET Listar todos los estudiantes

GET /server.php
No envía cuerpo. Devuelve la lista completa.

GET Buscar por ID

GET /server.php?id={id}
El ID viaja en la URL (query string).

POST Crear estudiante

POST /server.php
Los datos van en el cuerpo como JSON.

PUT Actualizar estudiante

PUT /server.php?id={id}
ID en la URL; datos completos en el cuerpo.

DELETE Eliminar estudiante

DELETE /server.php?id={id}
El ID va en la URL; sin cuerpo.
