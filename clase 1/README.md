# Clase 1 — Protocolo HTTP y Métodos

**Unidad 2: Interacción con el servidor**
**Sesión:** Teórica | **Temas:** 2.1 · 2.2

---

## Objetivos de la clase

- Comprender el ciclo petición-respuesta del protocolo HTTP
- Identificar y diferenciar los métodos HTTP: GET, POST, PUT, DELETE
- Asociar cada método con operaciones CRUD en una aplicación web real
